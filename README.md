## overview
this is a computational biology project exploring SEMA4A expression patterns as a training exercise in single-cell RNA-seq analysis. 

this project is a part of my efforts to develop my own R programming and statistical analysis skills while reproducing/extending analyses from published research.

## dataset - from _Cell Stem Cell_ (2024)
Jakobsen N, Turkalj S, Zeng A ...
Selective advantage of mutant stem cells in human clonal hematopoiesis is associated with attenuated response to inflammation and aging
Cell Stem Cell, 2024; 31, 1127-1144.e17

processed data obtained from the authors' Figshare repository. files used: TARGET-seq metadata, TARGET-seq processed raw counts

## what's being worked on currently
- exploratory analysis of SEMA4A expression
- expression by:
  - ell cluster
  - genotype
  - sample type
- distribution of expression
- mean expression across clusters
- cool figures using ggplot2 :D
